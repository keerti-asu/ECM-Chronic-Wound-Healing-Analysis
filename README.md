# ECM-Chronic-Wound-Healing-Analysis
Healthcare data analysis of ECM therapies for chronic wound healing using Python, Pandas, SciPy, and data visualization.

## Project Overview

Chronic wounds affect approximately 2.5% of the United States population and present major challenges due to infection risk, delayed healing, and comorbidities. Recent regenerative medicine research highlights the therapeutic potential of extracellular matrix (ECM) biomaterials derived from placental tissues, such as amnion and chorion membranes.

This project analyzes clinical research studies investigating ECM-based therapies for chronic wound repair. The analysis evaluates healing outcomes, treatment effectiveness, and clinical trends using data extracted from published trials.

The goal of this project is to demonstrate how data analytics and statistical methods can be applied to biomedical research to identify treatment patterns and therapeutic effectiveness.

Research Questions

1. Do ECM-based therapies improve wound healing time compared to standard treatments?
2. Are certain ECM types (amnion, chorion, ADSC-derived ECM) associated with higher healing success rates?
3. What clinical factors correlate with improved wound closure outcomes?

## Dataset

The dataset was curated from **published clinical studies on ECM-based wound healing therapies**. Variables include:

* Study name
* ECM type
* Number of patients
* Average healing time
* Healing success rate
* Control treatment outcomes

## Methods

The following analytical steps were performed:

* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Statistical comparison of ECM vs control treatments
* Visualization of healing outcomes
* Correlation analysis of clinical variables
* Hypothesis testing using t-tests

## Key Findings

* ECM therapies showed faster healing times compared to conventional treatments in multiple studies.
* Placental-derived ECM demonstrated higher wound closure rates**.
* Regenerative ECM therapies may reduce inflammation and improve tissue regeneration.
